# CreditLens Release Notes

## v0.1.0
- End-to-end scoring pipeline with logistic regression and random forest.
- Calibration, metrics reporting, and model metadata endpoints.
- SHAP explainability with what-if simulation support.
- Fairness diagnostics and drift monitoring summaries.
- Underwriter dashboard with applicants, fairness, monitoring, and model card views.
- One-command script to run tests and launch services.

## Notes
This is a demo release and is not intended for real lending decisions.
